Page   47: 3.36     Serial-wire JTAG debug port (SWJ-DP)
Page   47: The Arm SWJ-DP interface is embedded, and is a combined JTAG and serial-wire debug
Page   47: port that enables either a serial-wire debug, or a JTAG probe to be connected to the target.
Page   47: The debug is performed using two pins only instead of five required by the JTAG (JTAG pins
Page   47: can be reused as GPIO with alternate function): JTAG TMS and TCK pins are shared with
Page   47: switch between JTAG-DP and SW-DP.
